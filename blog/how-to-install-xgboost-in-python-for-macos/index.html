How to install XGBoost in Python for macOS? (2020.02)

February 1, 2021

2-Minute Read

Our goal is to install the XGBoost package, which can be compiled with multi-threads. Therefore, we need to install “GCC” which can compile multithread code. Here are the clear goals:

  1. Install GCC
  2. Install XGBoost package

What have already installed:

  1. Anacoda
  2. Python 3
  3. Homebrew

Installation

GCC

Obtain gcc from homebrew and check the version of gcc.

$ brew install gcc
$ gcc -dumpversion

Here, I was suggested by ref 2 to uncomment the following two lines in Makefile.

export CC = gcc
export CXX = g++

Then, add the following two lines in Makefile. (version of gcc)

export CC = gcc-12
export CXX = g++-12

XGBoost

Obtain libomp and cmake from homebrew.

$ brew install libomp # for multithread
$ brew install cmake # for cmake command below

change the directory to where you want to save xgboost folder and clone the repository.

$ cd <dir>
$ git clone --recursive https://github.com/dmlc/xgboost

Create the build/ directory and invoke CMake. After invoking CMake, you can build XGBoost with make:

$ cd <dir>
$ mkdir build # create a new and empty folder under <dir>
$ cd build # change directory to build
$ cmake .. # build several files automatically
$ make -j4 # specify the no. of cores

Install the XGBoost to your current Python environment.

$ python [setup.py](http://setup.py/) install

Finally, we can use conda list to check whether downloads successfully or not.

筆記

  1. Ref 2 安裝xgboost的方法不適用，所以找到 Ref 1 官方文件，其中有使用libomp，因此不確定gcc之必要性
  2. cmake 指令需要利用homebrew安裝 cmake(官方文件未註明)
  3. .. 代表上一層目錄；../.. 代表上上層目錄
